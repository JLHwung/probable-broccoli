# React + Redux Example App for Babel benchmark purpose

## Usage

Requires Node.js 12+

```
yarn
yarn profile
```

The CPU profile will be generated in `./cpuprofile`, please load the generated profile in the "Performance" Tab of Chromium DevTools.
